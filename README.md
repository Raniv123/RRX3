# 🔥 RRX3 v2.0

**אפליקציית קרבה אינטימית חכמה לזוגות עם AI Director**

---

## 🎯 מה זה RRX3?

אפליקציה מתקדמת לזוגות שמשלבת בינה מלאכותית חכמה עם תקשורת בזמן אמת, ליצירת חוויה אינטימית ייחודית ומותאמת אישית.

### ✨ תכונות עיקריות

- 🤖 **AI Director Mode** - בינה מלאכותית שחושבת על כל הודעה ומציעה את הצעד הבא
- 📊 **מד מתח חכם** - עוקב אחר המתח ומתקדם בצורה הדרגתית (ICE → WARM → HOT → FIRE)
- 🎭 **תרחישי Roleplay** - תרחישים דינמיים עם תפקידים, אישיויות ומבטאים
- ⏱️ **טיימרים חכמים** - למשחקים ופעילויות עם זמן מדויק
- 🎁 **מערכת הפתעות** - הפתעות אקראיות לפי רמת המתח (COMMON, RARE, EPIC, LEGENDARY)
- 🌍 **ביטויים בשפות זרות** - צרפתית, ספרדית, איטלקית
- 🔄 **סנכרון בזמן אמת** - שני המכשירים מחוברים באמצעות ntfy.sh
- 🎨 **עיצוב פוטוריסטי** - גלאסמורפיזם, צבעים חיים, אנימציות חלקות

## 🚀 התקנה מהירה

### דרישות מקדימות

- Node.js 18+ (המלצה: 20.x)
- npm או yarn
- חיבור אינטרנט

### שלבים

1. **שכפול הפרויקט:**
```bash
git clone <repository-url>
cd RRX3
```

2. **התקנת תלויות:**
```bash
npm install
```

3. **הגדרת מפתחות API:**

העתק `.env.example` ל-`.env`:
```bash
cp .env.example .env
```

פתח `.env` ומלא את מפתח ה-API של Google Gemini:
```
VITE_GEMINI_API_KEY=your_actual_key_here
```

**💡 איך להשיג מפתח Gemini API:**
- כנס ל-https://ai.google.dev
- לחץ על "Get API Key"
- צור פרויקט חדש או בחר קיים
- העתק את המפתח ל-`.env`

4. **הרצה:**
```bash
npm run dev
```

5. **פתח בדפדפן:**
```
http://localhost:5173
```

## 🧠 איך ה-AI עובד?

### ארכיטקטורה 4 שכבות

1. **Context Analysis** - ניתוח ההקשר הנוכחי (מצב רוח, רמת מתח, מספר הודעות)
2. **Progression Strategy** - החלטה מתי להאיץ/להאט, מתי לעבור שלב
3. **Recommendation Engine** - המלצות מדויקות למה לעשות עכשיו
4. **Adaptive Pacing** - קצב דינמי שמתאים את עצמו לפי התנהגות

### שלבי מתח (Phases)

| שלב | טווח | תיאור | דוגמאות פעולות |
|-----|------|-------|----------------|
| **ICE** ❄️ | 0-25% | היכרות, מגע עדין | מחמאות, מגע בכתף, חיוך |
| **WARM** 🔥 | 25-50% | התחממות, ביטויים | מגע אינטימי, מבטים, רמזים |
| **HOT** 🌶️ | 50-75% | פעולות נועזות | dirty talk, אוראלי, טיימרים |
| **FIRE** 💥 | 75-100% | עוצמה מקסימלית | ביטויים מלוכלכים, קיצוניות |

### טיימרים

ה-AI מציע פעולות עם טיימרים מדויקים:
- **אוראלי לגבר** (על ידי אשה): 1:20 (80 שניות)
- **אוראלי לאשה** (על ידי גבר): 2:00-2:35 (120-155 שניות)
- **מגע אינטנסיבי**: 1:00-1:30
- **Tease/Warmup**: 2:00

## 🎁 מערכת הפתעות

### סוגי הפתעות

- **DARE** - מעשים נועזים (Secret Touch, Whisper Challenge)
- **GAME** - משחקים (Yes Master, Freeze Game, Mirror Me)
- **CHALLENGE** - אתגרים (Language Switch, No Hands, Eyes Closed)
- **TWIST** - עלילות פתע (Phone Ring, Neighbor Alert, Lights Out)

### נדירות

- **COMMON** (10x weight) - מופיעות לעתים קרובות
- **RARE** (5x weight) - מיוחדות
- **EPIC** (2x weight) - נדירות
- **LEGENDARY** (1x weight) - מאוד נדירות!
